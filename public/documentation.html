JMD MOVIE DATABASE API DOCS

Welcome to the JMD API Docs! This API stores data on a variety of movies. The URL endpoints below outline all possible requests.

METHOD & URL

FUNCTIONALITY

REQUEST BODY FORMAT

RESPONSE BODY FORMAT (ex.)

GET /movies

Returns a list of ALL movies.

N/A

JSON Object {"Movies" : [List]}

GET /movies/:title

Returns data about a single movie.

N/A

JSON Object {"Movies" : {"title" : "data"}}

GET /genres/:genreName

Returns data about a genre by name.

N/A

JSON Object {"genreTitle": "desc"}

GET /directors/:directorName

Returns data about a director by name.

N/A

JSON Object {"directorName" : "data"}

POST /users

Creates a user account.

JSON Object containing a username and password.
{"username" : "data",
"password" : "data"}

Text confirmation that the new user has been created with the given credentials.

PUT /users/:id

Allows user to update profile by ID.

N/A

Text confirmation that the user's ID has been updated to the given value.

POST /users/:id/:title

Adds a movie to a user's favorites list.

N/A

Text confirmation that the indicated movie has been added to the user's favorites list.

DELETE /users/:id/:title

Removes a movie from a user's favorites list.

N/A

Text confirmation that the indicated movie has been removed from the user's favorites list.

DELETE /users/:id

Deletes a user account.

N/A

Text confirmation that the indicated user account has been successfully deleted.
